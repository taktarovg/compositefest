# Обновление на GitHub Pages

Выполните эти команды в терминале (PowerShell или Git Bash):

```bash
# 1. Перейти в папку проекта
cd C:\dev\composit

# 2. Добавить все изменения
git add .

# 3. Сделать коммит
git commit -m "Добавлены демо-страницы для всех разделов сообщества"

# 4. Загрузить на GitHub
git push
```

Готово! Сайт обновится через 1-2 минуты.
